# FoodProj
This project uses spoonacular api (https://spoonacular.com/food-api) to provide the user with access to various dishes. 

Website tabs:
- "Random Recipes" tab: shows a random dish with detailed information such as ingedients and instructions.
- "Search Recipes" tab: lets you search for a dish by name (autocomplete search) and gives short description of each dish and important information that matches your search.

To run the project:
- Download files
- Open them in vscode
- Type "npm i" in the terminal (this will install all packages that are used in this project
- To run the code type "node index.js" in the terminal (note: nodeJS must be installed on your system. Go to https://nodejs.org/en/download if you dont have it)
- Open your browser and go to http://localhost:3000/
